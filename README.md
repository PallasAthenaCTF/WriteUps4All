# WriteUps4All
CTF  writeups

# Structure
For each event/site one single main folder named after the event/site. Here on short Readme.md with information like links, event dates etc.
There should be subfolders for each category, e.g. web, pwn, rev etc.
These subfolders should contain folders for each challenge, which contain the writeup and needed files. 

## Example
Folders are mark bold and files iltalic.

* **OverTheWire**
  * _Readme.md_
  * **Bandit**
    * **bandit0**
      * _Writeup.md_
